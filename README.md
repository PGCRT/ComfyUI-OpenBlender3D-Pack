# ComfyUI OpenBlender 3D Pack

3D nodes repacked from:

- ComfyUI-MotionCapture https://github.com/PozzettiAndrea/ComfyUI-MotionCapture
- ComfyUI-SAM3DBody https://github.com/PozzettiAndrea/ComfyUI-SAM3DBody
- ComfyUI-SAM3DObjects https://github.com/PozzettiAndrea/ComfyUI-SAM3DObjects
- ComfyUI-TRELLIS2 https://github.com/PozzettiAndrea/ComfyUI-TRELLIS2
- ComfyUI-HY-Motion https://github.com/PozzettiAndrea/ComfyUI-HyMotion

This pack is intended to replace separate installation of those nodes for the
OpenBlender AIO flow.

## Install

Run from this directory with the ComfyUI embedded Python:

```powershell
python install.py
```

Restart ComfyUI after installation.
